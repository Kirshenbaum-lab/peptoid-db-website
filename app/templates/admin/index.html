PeptoidDB Admin

Peptoid Data Bank - Administration portal

Here is the portal for administering the Peptoid Data Bank. Administrators have access to views titled Peptoid, Residue, and Author which will allow the administrator to create, read, update, and delete peptoids, residues, and authors from the database. In addition to these functions for adding in new published data, it is important to add images of any new peptoid crystal structures or residues in order to maintain the appearance of the website, which is why administrators can do just that by uploading .png images after clicking on Peptoid Images or Residue Images in the navigation bar on top. For major updates that cannot be made in this administrative portal please visit the GitHub, read the README.md, clone the code, and make changes. Manage your versions using Git and deploy the new application to the server by following the instructions in the README.md. Happy administering!
